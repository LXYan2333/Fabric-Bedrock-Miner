# 使用
准备好如下物品：
1. 效率 Ⅴ 钻石镐（下界合金也行） 
2. 急迫 Ⅱ 信标
3. 一些活塞
4. 一些红石火把
5. 黏液块若干

使用 `左 ALT+B+C` 快捷键打开配置菜单

使用 `左 ALT+B+M` 开启/关闭本 mod

本 mod 开启状态下，左键单击

所有快捷键均可配置

如果本 mod 帮你节省了大量时间的话，给个 Star 呗 qwq。
